# JAMIS 2.0 Prototype

High-fidelity React prototype for the Jigawa Agricultural Management Information System.

## Stack
- Vite + React
- Tailwind CSS
- Lucide Icons
- Recharts
- Framer Motion

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` to view the War Room.
